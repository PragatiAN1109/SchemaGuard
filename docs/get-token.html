SchemaGuard — Google ID Token Demo

Sign in with Google to get an RS256 ID token you can use to call the API.

How to use this token: 
# 1. Copy the token from the box above

# 2. Call the API (replace TOKEN with your copied token):
curl -X POST http://localhost:8080/api/v1/plan \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer TOKEN" \
  -d @samples/plan.json

# 3. Verify it's RS256 — paste the token at https://jwt.io
#    You'll see: "alg": "RS256" in the header section

# 4. No token → 401:
curl http://localhost:8080/api/v1/plan/some-id
# returns: 401 Unauthorized

# 5. Schema endpoint is public (no token needed):
curl http://localhost:8080/api/v1/schema/plan
# returns: 200 OK with the JSON Schema
